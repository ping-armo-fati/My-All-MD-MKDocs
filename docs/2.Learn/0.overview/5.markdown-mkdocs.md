## Build

- Edit the `.readthedocs.yaml`

```yaml
# refer from: https://docs.readthedocs.io/en/stable/config-file/index.html
version: 2
build:
  os: ubuntu-22.04
  tools:
    python: "3.12"
mkdocs:
  configuration: mkdocs.yml
python:
  install:
  - requirements: docs/requirements.txt
```
