# 一常用快捷键
==============

功能快捷键

|            |                          |
|------------|--------------------------|
|            |                          |
| 格式刷     | ctrl + shift + p         |
| 指针工具   | ctrl + 1                 |
| 文本工具   | ctrl + 2                 |
| 连接线工具 | ctrl + 3                 |
| 组合所选项 | ctrl + g（group）        |
| 取消组合   | ctrl + shift + u（undo） |
| 置于顶层   | ctrl + f（foreground）   |
| 置于底层   | ctrl + b（background）   |
| 移动所选项 | 方向键                   |
| 按像素移动 | shift + 方向键           |
